# 노션 Embed 가이드

## 🎯 노션에 웹 앱을 삽입하는 방법

### 1단계: 애플리케이션 배포
먼저 웹 애플리케이션을 배포해야 합니다. [DEPLOYMENT.md](DEPLOYMENT.md) 참고

### 2단계: 노션에서 Embed 블록 추가

#### 방법 A: 명령어 사용
1. 노션 페이지를 엽니다
2. 새 줄에서 `/embed` 또는 `/임베드`를 입력합니다
3. 배포된 URL을 붙여넣습니다
   ```
   https://your-app.railway.app
   ```
4. "Embed link" 버튼을 클릭합니다

#### 방법 B: URL 직접 붙여넣기
1. 노션 페이지에 배포 URL을 직접 붙여넣습니다
2. 팝업에서 "Create embed" 옵션을 선택합니다
3. 자동으로 embed 블록이 생성됩니다

### 3단계: 크기 조정
- Embed 블록의 가장자리를 드래그하여 크기를 조정할 수 있습니다
- **Full width** (전체 너비) 추천

## ⚙️ Embed 설정 팁

### Full Width로 설정
1. Embed 블록 클릭
2. 6개 점 메뉴 (⋮⋮) 클릭
3. "Full width" 선택

### 높이 조정
- Embed 블록 하단을 드래그하여 높이 조정
- 권장 높이: 600-800px

## 🚨 Embed 문제 해결

### "This content cannot be displayed" 오류

**원인:**
- HTTP URL 사용 (HTTPS 필요)
- 서버가 X-Frame-Options 헤더로 iframe 차단
- 배포 서버가 다운됨

**해결 방법:**
1. HTTPS URL 사용 확인
2. Railway/Render 등 embed를 지원하는 플랫폼 사용
3. Bookmark 대신 사용

### Embed가 너무 작게 표시됨

**해결 방법:**
1. Full width 설정
2. Embed 블록 하단 드래그로 높이 조정
3. 웹 앱 자체에서 min-height CSS 설정

### 모바일에서 작동하지 않음

**원인:**
- 노션 모바일 앱에서 Embed 지원 제한

**해결 방법:**
- Bookmark 링크 사용
- 사용자가 클릭하면 브라우저에서 열림

## 🔗 공유하기

### 노션 페이지 공유
1. 우측 상단 "Share" 클릭
2. "Share to web" 토글 활성화
3. 생성된 링크를 다른 사람들과 공유
4. 다른 사용자들도 embed된 앱을 사용할 수 있습니다!

### 권한 설정
- **Can view**: 보기만 가능 (embed 앱은 사용 가능)
- **Can comment**: 댓글 추가 가능
- **Can edit**: 편집 가능

## 💡 고급 활용

### 여러 페이지에 동일한 Embed 사용
- 한 번 배포한 URL을 여러 노션 페이지에 embed 가능
- 중앙에서 업데이트하면 모든 embed에 자동 반영

### 팀 워크스페이스 공유
1. 팀 노션 워크스페이스에 페이지 생성
2. Embed 추가
3. 팀원 모두 사용 가능

### 데이터베이스에 추가
- 노션 데이터베이스의 페이지 내부에 embed 가능
- 각 프로젝트/작업마다 도구 제공
